# Juego de Operaciones Matemáticas (Python)

Juego por consola desarrollado en Python donde el jugador debe resolver
operaciones matemáticas para avanzar de nivel. El juego incluye un sistema
de vidas, niveles progresivos y validación de respuestas.

## Cómo usar el juego

Requisitos:
- Python 3

Ejecución:
1. Abrir el símbolo del sistema (cmd)
2. Buscar la carpeta donde se haya guardado el archivo
3. Ejecutar el programa con python3
4. python3 juego_sumas.py

## Funcionamiento del juego
- El jugador comienza con un número limitado de vidas
- Cada nivel presenta una operación diferente:
  - Suma
  - Resta
  - Multiplicación
  - División (con decimales)
- Si la respuesta es correcta, se avanza de nivel
- Si la respuesta es incorrecta, se pierde una vida
- El juego termina cuando se completan los niveles o se quedan sin vidas

## Tecnologías usadas
- Python 3
- Librería random
- Librería sys

## Habilidades aprendidas
- Uso de bucles (`while`)
- Validación de datos introducidos por el usuario
- Manejo de números enteros y decimales
- Uso de `try / except` para control de errores
